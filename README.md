# Yield Browser v0.9.72

Source revisi Yield Browser v0.9.72.

Fokus update:
- kontrol video portrait lebih bersih tanpa toast yang menutup tombol,
- tombol Play/Pause digabung jadi satu tombol toggle,
- fullscreen video dibuat lebih responsif dengan fallback otomatis,
- kontrol video fullscreen/landscape dibuat lebih lega, lalu kembali compact saat portrait,
- YouTube tombol Lewati diperkuat dengan native tap bridge dari WebView agar klik lebih mirip tap asli.

Fitur v0.9.71 tetap dipertahankan: YouTube auto cycle ad bypass, edge gesture guard, universal blank compatibility, desktop/mobile fix, search engine fix, Lordborg, Instant Monitor, Invest-tracing, direct image guard, night mode, translate, dan download manager.
