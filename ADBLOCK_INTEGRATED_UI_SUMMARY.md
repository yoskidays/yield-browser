# AdBlock Integrated UI — v0.10.00

Shield Engine V2 tetap digunakan sebagai mesin internal, tetapi tidak ditampilkan sebagai fitur atau menu terpisah.

Perubahan antarmuka:
- Shortcut menu utama kembali bernama `AdBlock ON/OFF`.
- Baris pengaturan kembali bernama `AdBlock: ON/OFF`.
- Dialog pengaturan kembali berjudul `AdBlock`.
- Pengaturan kustomisasi shortcut kembali memakai nama `AdBlock`.
- Tidak ada statistik blokir, badge, panel Shield terpisah, atau toast rutin.

Tidak ada perubahan pada kebijakan pemblokiran Shield Engine V2. Perubahan ini hanya menyatukan kembali seluruh kontrol ke tombol AdBlock yang sudah ada.
