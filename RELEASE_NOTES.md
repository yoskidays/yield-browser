# Yield Browser v0.9.49

## Rilis Terbaru
Versi ini merapikan kontrol video dengan tombol mode tunggal.

## Perubahan Utama
- Tombol `Land` dan `Full` digabung menjadi satu tombol.
- Saat tidak fullscreen, tombol tampil `Full` untuk masuk ke Full Video.
- Saat sedang Full Video, tombol berubah menjadi `Lans` untuk kembali ke Landscape Video biasa.
- Kontrol video tetap dipulihkan saat keluar dari Full Video agar tidak hilang.
- Tidak mengubah fitur yang sudah stabil: Desktop/Mobile mode, adblock, translate, night mode, direct image guard, download manager, dan site compatibility.

## Catatan
Release note lama sudah dihapus agar folder project lebih rapi. Mulai versi ini cukup gunakan file `RELEASE_NOTES.md` sebagai catatan rilis terbaru.
